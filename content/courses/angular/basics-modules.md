Get comfortable with declarations, imports, and exports.

{{< file "ngts" "some.module.ts" >}}

```typescript
@NgModule({
  // Components defined in this module
  declarations: [
    FooComponent
  ],
  // Components used in this module
  imports: [
    CommonModule
  ],
  // Components that can be imported by other modules
  exports: [
    FooComponent
  ],
})
```
